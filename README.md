# Spoken Partner (PWA)

এই ফোল্ডারে আপনার আগের অ্যাপটি এখন **Install App** সাপোর্টসহ একটি Progressive Web App (PWA)।

## ভেতরে যা আছে
- `index.html` — মূল অ্যাপ (নাম পরিবর্তন করে **Spoken Partner** করা হয়েছে + PWA ট্যাগ যোগ হয়েছে)
- `manifest.json` — অ্যাপের নাম, আইকন, থিম কালার ইত্যাদি
- `service-worker.js` — অফলাইন সাপোর্ট (অ্যাপ শেল ক্যাশ করে রাখে)
- `icons/` — অ্যাপ আইকন (192px, 512px, ও maskable ভার্সন)

## GitHub-এ কীভাবে আপলোড করবেন
1. আপনার GitHub রিপোজিটরির পুরোনো ফাইলগুলো মুছে দিন (বা এই ফাইলগুলো দিয়ে ওভাররাইট করুন)।
2. এই ফোল্ডারের **সব ফাইল ও `icons/` ফোল্ডার সহ** রুটে আপলোড করুন — ফোল্ডার স্ট্রাকচার ঠিক রাখা জরুরি।
3. রিপোজিটরিতে GitHub Pages চালু থাকলে কিছুক্ষণের মধ্যেই লাইভ হয়ে যাবে।

## গুরুত্বপূর্ণ শর্ত (Install অপশন আসার জন্য)
Chrome/Edge-এ "Install app" অপশন আসতে হলে সাইটটি অবশ্যই:
- **HTTPS**-এ হোস্ট হতে হবে (GitHub Pages এমনিতেই HTTPS দেয় ✅)
- `manifest.json` ও `service-worker.js` সঠিকভাবে লোড হতে হবে (ফাইল পাথ ঠিক রাখতে হবে)

এই শর্তগুলো পূরণ হলে ব্রাউজারের ঠিকানার বারে বা মেনুতে **Install** আইকন/অপশন দেখা যাবে। ইউজার
চাইলে হেডারের **"⬇️ অ্যাপ ইনস্টল করুন"** বাটনেও ট্যাপ করতে পারবেন।

## অন্যরা কীভাবে ইনস্টল করবে
আপনার GitHub Pages লিংক (বা যেকোনো হোস্টিং লিংক) কাউকে দিলে, তারা:
- **Android/Chrome**: লিংকে ঢুকে মেনু (⋮) থেকে "Install app" / "Add to Home screen" চাপবে, অথবা হেডারের ইনস্টল বাটনে ট্যাপ করবে।
- **iPhone/Safari**: Share বাটন থেকে "Add to Home Screen" বেছে নেবে (iOS-এ স্বয়ংক্রিয় install prompt দেখায় না, এটাই Apple-এর সিস্টেম)।
- **Desktop Chrome**: ঠিকানার বারে ডানদিকে একটি ইনস্টল (⊕/মনিটর) আইকন আসবে।

ইনস্টলের পর অ্যাপটি আলাদা উইন্ডোতে (ব্রাউজার UI ছাড়া) খুলবে এবং হোম স্ক্রিন/অ্যাপ লিস্টে
**Spoken Partner** নামে ও নতুন আইকনসহ দেখাবে। অনুবাদের জন্য (MyMemory API) ইন্টারনেট লাগবে,
তবে অ্যাপ ইন্টারফেসটি একবার খোলার পর অফলাইনেও চালু হবে।
